### 볼륨

* 공유 방법
  * volumeMounts
* 유형

  * hostPath : Local Disk에 생성
  * gitRepo : emptyDir에 git clone
  * PersistentVolume (PV) : 정적 프로비져닝, pod와 별개로 운영
  * PersistentVolumeClaim (PVC) : 동적 프롭져닝, 사용자가 PV에 하는 요청
