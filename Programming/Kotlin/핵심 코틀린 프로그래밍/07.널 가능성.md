## 널 도입과 널 가능성의 필요성
#### 

```kt
fun <T> find(array: Array<T>, predicate: (T)-> Boolean) : T {

    for(element in array) {
        if (predicate(element)) return element
    }
    throw NotFound()
}
```

```kt
fun <T> find(array: Array<T>, predicate: (T)-> Boolean) : T {

    for(element in array) {
        if (predicate(element)) return element
    }
    return null
}
```
