### Redis의 정의
* Remote Dictionary Server
* Storage : 데이터 저장소 (데이터 관점)
* Database : 전통적인 DBMS의 역할을 수행 (영속성 관점)
* Middleware : 어플리케이션이 이용할 수 있는 유용한 기능을 제공하는 SW

<br>

### Redis로 할 수 있는 것?
* 아주 빠른 데이터 저장소로 활용
* 분산된 서버들 간의 커뮤니케이션
* 내장된 자료구조를 활용한 기능 구현
