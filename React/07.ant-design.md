## 기본 사용법

#### 

### [.js File 최상단]
```js
import { Layout, Menu, Spin } from 'antd';
import "antd/dist/antd.css";
```
* 위의 형태로, `script` 태그와 `link` 태그를 import해서 사용하는 형식
  * `antd`에서 Component를 Import해서 바로 사용
