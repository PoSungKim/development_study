## Installation

#### React 라이브러리는 SPA 구현을 위해서 필요한 ```URL, Component, Browser History``` 관리까지는 built-in 형식으로 해주지 않기 때문에, ```React Router DOM``` 라이브러리를 추가적으로 설치해야 한다.

<br>

### [Terminal]
```bash
yarn add react-router-dom 
```
