## SessionStorage
#### 임시 저장
#### HTML 5부터 제공

<br>

### [예시]
```ts
console.log(window.sessionStorage);
console.log(sessionStorage);
sessionStorage.setItem("key", "value");
sessionStorage.getItem("key");          // value
```
