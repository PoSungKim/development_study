### [package >> node_modules >> package-lock]
* 동일한 package에는 동일한 node_modules가 생성되는 것이 이상적
* 다만, 동일하지 않는 경우가 발생하는데, 다음과 같은 이유
  * 현재 사용하고 있는 npm의 버전이 다른 경우
  * 의존성을 가진 패키지의 버전이 업데이트 됐을 경우
  * 의존성을 가진 패키지가 의존하는 패키지의 버전이 업데이트 됐을 경우
