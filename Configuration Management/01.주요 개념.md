## 1. 중앙집중관리식의 SVN

#### Remote Repository <==> Local PC 

### [Terminal]
```bash
check-out
update
commit
```
* 명령어를 실행하면 배포 관련 안정성 체크를 하고 부족한 점들을 안내해줌

<br>
<hr>

## 2. 분산관리식의 GIT

#### Remote Respository <==> Local Repository <==> Local PC

### [Terminal]
```bash
git push
git pull
git branch 
git commit
```
* 명령어를 실행하면 배포 관련 안정성 체크를 하고 부족한 점들을 안내해줌
