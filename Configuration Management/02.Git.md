# Git

<br>

## local

```
Working Directory
- untracked files
- tracked files

Staging Area

Git Repository

```

* Tracked Files : Staging Area에 없는 파일

* Tracked Files : Staging Area에 있지만, 내용이 달라진 파일

* `git rm --cached` : Staging Area에 있는 파일 정보들을 초기화
  * 즉, 모든 tracked files가 untracked files가 된다

* `.gitignore` : Git Repository에 올리지 않을 파일 이름 추가
  * 정규식으로 이름 표현 가능
