## sbt
####

<br>

### [Command]
```bash
sbt           # sbt server 접속 및 Dependencies .jar 다운로드

sbt > compile  # 컴파일 
sbt > run      # main 함수 실행
sbt > package  # .jar 파일 빌드
sbt > help     # 
sbt > assembly # fat jar
```
