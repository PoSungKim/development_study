# Elastic Load Balancing
> 미리 정해 놓은 규칙에 따라 워크로드를 자동으로 확대 또는 축소할 수 있는 기술로 클라우드가 제공하는 탄력성에 의해 만들어지고, 사용자의 요구 수준을 세심하게 반영할 수 있는 기술
* Peak Workloads에 맞춰 충분한 리소스 확보 가능
