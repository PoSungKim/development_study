### [IAM 설정]
* IAM 페이지 : EC2 대상으로 AdministratorAccess 역할 생성
* EC2 페이지 : 작업 > 보안 > IAM 역할 수정

<br>

### [Bucket 생성]
```bash
aws s3 mb s3://${bucketname} --region ap-northeast-2
```

<br>

### [Bucket에 파일 전송]
```bash
aws s3 sync . s3://${bucketname} --exclude '.git/*'
```

<br>

### [Bucket에서 수신]
```bash
aws s3 cp s3://${fromDir} ${toDir} --recursive
```
