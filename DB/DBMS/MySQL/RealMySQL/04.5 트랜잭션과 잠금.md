# 트랜잭션과 잠금
> MySQL의 동시성에 영향을 미치는 잠금(Lock)과 트랜잭션, 트랜잭션의 격리 수준 (Isolation Level)
* 잠금을 통해 동시성 제어 
* 트랜잭션을 통해 작업의 완전성과 데이터 정합성 보장 (Partial Update X)
* 격리 수준은 하나의 트랜잭션 내에서 또는 여러 트랜잭션 간의 작업 내용을 어떻게 공유하고 차단할 것인지 결정
