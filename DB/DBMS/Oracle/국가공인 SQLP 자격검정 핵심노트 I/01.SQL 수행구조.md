### 오라클 메모리 캐시
* DB 버퍼캐시 : 테이블 블록, 인덱스 블록, Undo 블록
* 라이브러리 캐시 : SQL, DB 저장형 함수/프로시저, 트리거
* 딕셔너리 캐시 : 테이블 정보, 인덱스 정보, 데이터파일 정보, 시퀀스
* Result 캐시 : SQL 결과집합
