## NL 조인
#### 

<br>

### ordered vs leading
* ordered use_nl(c) use_hash(t)
* leading(o c t) use_nl(c) use_hash(t)

<br>

### ordered use_nl(c) index(e) index(c)
* NL 조인 데이터 엑세스 순서
  * index - table - index - table
```sql
select /*+ ordered use_nl(c) index(e) index(c) */
    ...
from 사원 e, 고객 c
where 1=1
and e.c = c.c
and e.a >= '19960101'
and e.b = 'Z123'
and c.d >= 20000
```
