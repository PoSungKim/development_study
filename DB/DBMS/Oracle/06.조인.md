### (+) Outer Join
* (+)이 붙는 쪽에 Null 값이 들어간다
```sql
SELECT a.col1, b.col1
FROM   tab1 a, tab2 b
WHERE  a.col1 = b.col1(+)
;

SELECT a.col1, b.col1
FROM   tab1 a RIGHT OUTER JOIN tab2 b
ON     a.col1 = b.col1
;
```

```sql
SELECT a.col1, b.col1, c.col1
FROM   tab1 a, tab2 b, tab3 c
WHERE  a.col1 = b.col1(+)
  AND  b.col1 = c.col1(+);
;

-- 둘 다 상단의 예시처럼 Right Outer Join
```
