## 쿼리 변환 대상 최소 단위
#### Query Block

<br>

### Query Block
* Type
  * Top-Level Statement
  * Subquery
  * Unmerged View
* 예시) 2 Query blocks
  * outer query block = Top-Level Statement
  * inner query block = Subquery
  ```
  SELECT first_name, last_name
  FROM   hr.employees
  WHERE  department_id 
  IN     (SELECT department_id 
          FROM   hr.departments 
          WHERE  location_id = 1800);
  ```

<br>

### Query Subplan
* For each query block, the optimizer generates a query subplan
