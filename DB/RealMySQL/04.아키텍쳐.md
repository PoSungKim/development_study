<br>

### 스토리지 엔진
* MySQL 서버 : 1개의 MySQL 엔진 + N개의 스토리지 엔진으로 구성 가능
```sql
CREATE TABLE test_table (fd1 INT, fd2 INT) ENGINE=INNODB;
```
* 각 스토리지 엔진은 성능 향상을 위해 키 캐시 (MyISAM 스토리지 엔진) 혹은 InnoDB 버퍼 풀 (InnoDB 스토리지 엔진)과 같은 기능을 내장하고 있다
