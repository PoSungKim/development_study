## 매개 변수 변경

#### 매개변수 : 상수를 변수로 만들어서 화면에서 동적으로 변화되는 값
#### 사용자가 Dashboard에서 매개변수를 변경함으로써 그래프가 동적으로 변경된다

### [주요 개념]

```bash
1. 매개 변수 생성
2. 매개 변수가 취할 수 있는 다양한 값에 대한 색상 설정
3. 매개 변수가 취할 수 있는 값 각각에 대한 계산 필드를 생성
4. 계산 필드의 리턴 값이 연속형이 될 수도 있고 불연속형이 될 수도 있기 때문에, 연속형일 경우에는 합계 등을 계산해서 그래프를 그릴 수 있는 필드로 사용 가능
5. 색상은 매개 변수, 그래프는 계산 필드로 연결
6. 동작 > 매개 변수 > 대상 매개 변수의 값에 필드의 값을 주입시켜준다
  - 즉, 매개 변수가 취할 수 있는 값과 주입하고자 하는 필드의 값이 서로 맞아떨어져야 한다
  - ex) 필드 값인 어떠한 시트의 `측정값 이름`이 대상 매개 변수인 `P_측정값 선택`의 옵션들 중에 하나의 값을 취해야만 한다
```
* 가장 주요 개념 : `색상은 매개 변수`, `그래프는 계산 필드`로 연결
