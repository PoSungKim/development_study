# Header 정보


### [ResponseEntity, HttpHeaders]
```java
public ResponseEntity<HashMap<String, String>> addNewUser(HashMap<String, String> Message) {
        HttpHeaders responseHeaders = new HttpHeaders();
        responseHeaders.add("meta-chatbot", "true");
        Message.put("content", ChatBot.sayHello());
        Message.put("userName", String.format("손님%d\n", ChatBot.getUserNum()));
        return ResponseEntity.ok().headers(responseHeaders).body(Message);
    }
```
