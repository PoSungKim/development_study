## 1. Array 배열

#### Primitive Data Type으로 기초적인 자료구조고, Byte 크기가 정해져 있고, Time Complexity가 

<br>

### [선언]
```java
int arr[] = new int[10];
int[] arr = new int[10];
```

<br>

### [초기화]
```java
int[] numbers = new int[] {10, 20, 30};
int[] numbers = {10, 20, 30};

int[] numbers;
numbers = new int[] {10, 20, 30};
```

<br>

### [Enhanced For Loop]
```java
for(int num : numbers) {
  System.out.println(num);
}
```
