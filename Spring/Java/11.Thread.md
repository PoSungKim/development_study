## Thread
> 프로그램이 실행되면 OS로 부터 메모리를 할당받아 프로세스 상태가 됌
* 프로세스는 하나 이상의 Thread를 가지게 되고, 실제 작업을 수행하는 단위는 Thread

```java
public abstract class Decorator extends Coffee {
  Coffee coffee;
  
  public Decorator (Coffee coffee) {
    this.coffee = coffee;
  }
  
  @Override
  public void brewing() {
    coffee.brewing();
  }
}
```
