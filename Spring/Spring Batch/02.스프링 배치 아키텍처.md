### [Job]

* ```JobLauncher```에 의해 실행 
* 배치의 실행 단위
* N개의 Step 실행 가능 (Flow 제어 가능)

<br>

### [Step]

* Job의 세부 실행 단위이며, N개가 등록되어 실행됌
* 실행 단위
  * Chunk 기반 : 하나의 큰 덩어리를 n개씩 나눠서 실행
  * Task 기반 : 하나의 작업 기반으로 실행
