## Hibernate & JPA (ORM) 
#### ORM : SQL문 작성할 필요없이 Java만으로 DB 접속
#### JPA : DB접속을 위한 명세
#### Hibernate : JPA의 구현체
#### JPA도 내부적으로 JDBC 사용

<br>

### [예제]
```java
interface TestRepository extends JpaRepository<Test, Long> {
}
```
* JpaRepository가 JPA Interface

* 상속 Rule  
  * `Interface extends Interface`
  * Interface cannot extends Class
  * Class extends Class
  * Class implements Interface

<br>
