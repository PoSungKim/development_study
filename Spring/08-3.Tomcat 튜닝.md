# Tomcat 튜닝
> Tomcat을 WAS로 사용 시, Thread Pool 관리를 통해 인프라 개발자의 서버 안정성 그리고 어플리케이션 개발자의 서버 성능 등을 함께 잡을 수 있다
*

## Spring Boot 기준 application.yml
#### 

<br>

### ThreadPoolExecutor
```yml
server:
  tomcat:
    threads:
      max: 200
      min-spare: 10
    accept-count: 100
```
