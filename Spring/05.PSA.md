## Spring WebFlux
#### 자동으로 내부 Netty로 실행

<br>

### [키포인트]
* Spring 5부터 사용 가능한 WebFlux, Web과 모든 코드가 호환되는 것은 아니고, 내부적으로 동작하는 메커니즘이 다르다
* Spring Web MVC vs Spring WebFlux
  * Web의 경우, Request 당 Thread가 생성되어 수행
  * WebFlux의 경우, Thread의 개수를 CPU의 개수와 같이 유지하면서 보다 효율적으로 수행
